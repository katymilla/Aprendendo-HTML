Características do HTML

(Voltar)

HTML (Hypertext Markup Language) é uma linguagem para estruturar layouts das páginas da web, podendo ser executada diretamente no navegador. Não é considerada uma linguagem de programação, pois não pode realizar algumas funções. As linguagens de programação consistem em liguagens escrita e formal que especificam um conjunto de instruções e regras usadas para gerar programas (softwares). As cinco primeiras linguagens de programação são: JavaScript, Python, Java, PHP e C#. O HTML faz parte da tríade HTML, CSS e JavaScript.

Estrutura do HTML

(Voltar)

As três estruturas básicas do HTML são: html, head e body. O HTML é toda a estrutura, o head é o que vai carregar antes de aparecer para o usuário e o body é o que aparece na página para o usuário.

Como iniciar um projeto no VS Code com HTML?

(Voltar)

  1. Instalar VS Code;
  2. Criar uma pasta no seu computador com o nome do projeto;
  3. Abrir o VS Code;
  4. File-Open Folder-escolher a pasta que foi criada;
  5. Após a pasta ter sido aberta no VS Code clicar em open file e criar um arquivo com o nome index.html;
  6. No arquivo index.html digitar html:5 que vai abrir toda a estrutura do html;
  7. Para salvar o projeto no VS Code você pode digitar Ctrl+S;

Algumas tags relevantes no HTML

(Voltar)

A sintaxe é a forma como é codificado algo e é na forma de tags no HTML. As tags são a estrutura do site e elas estão dentro dos sinais <>. Existem as tags de fechamento e as tags sem fechamento. Segue abaixo algumas tags relevantes no HTML.

  1. p: parágrafo;
  2. h1-h6: representa a importância dos títulos e o tamanho deles na página;
  3. footer: rodapé da página;
  4. section: sessão para a página;
  5. header: cabeçalho da página;
  6. article: conteúdo relevante;
  7. aside: conteúdo relacionado ao conteúdo principal;
  8. div: representa uma divisão da página;
  9. input: é uma tag para abrir uma caixa de texto e não tem a tag de fechamento. Essa tag é muito utilizada para elaboração de formulários;
  10. img: é uma tag relacionada à imagem e também não tem fechamento;
  11. mark: marca o texto com uma cor diferente;
  12. u: sublinha o texto;
  13. i: deixa o texto em itálico;
  14. strong: deixa o texto em negrito;
  15. blockquote: é uma citação de outro autor ou uma observação que deixará o texto com espaçamento;
  16. ol: lista ordenada;
  17. li: é o item da lista;
  18. ul:lista não ordenada;
  19. button: para adicionar um botão;
  20. sup: deixa o texto sobrescrito;
  21. sub: deixa o texto subscrito;
  22. hr: coloca uma linha embaixo do texto;
  23. iframe: cria uma janela que mostra conteúdo de outra página, por exemplo, uma janela com um vídeo de outro site;
  24. textarea: cria uma caixa de texto;
  25. fieldset: cria uma segmentação de uma área dentro da página;
  26. small: reduz o tamanho da fonte de um texto;
  27. a: é uma âncora, para criar vínculo com outro documento;
  28. font: utilizado para configurar a fonte, a cor e o tamanho das letras de um texto;
  29. del: representa uma parte do texto que foi excluída de um documento;
  30. abbr: indica abreviatura ou sigla;

Atributos das tags

(Voltar)

Atributos são opções especiais para personalizar determinadas tags. Segue abaixo alguns atributos relevantes no HTML.

  1. id: identificador do elemento;
  2. style: voltado para o CSS;
  3. type: específico do input;
  4. src: específico do img, ele é obrigatório e guarda o caminho da imagem;
  5. alt: utilizado junto com img, não é obrigatório, mas é recomendado por mostrar a descrição da imagem;
  6. width: utilizado junto com a imagem para determinar seu tamanho;
  7. class: atribui uma classe ao elemento;
  8. title: atribui um título ao elemento;
